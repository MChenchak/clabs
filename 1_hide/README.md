Утилита hide создает "темную" директорию в текущем каталоге и перемещает в нее указанный файл.
Если "темная" директория уже создана, то перемещает в нее указанный файл. 

### Compilation info:

```
1. Clone repository
2. cd clabs/1_hide
3. mkdir build
4. cd build
5. cmake ..
6. make
```

### Usage

```
Syntax: ./hide <path>

Examples:
        ./hide 1.txt
```
