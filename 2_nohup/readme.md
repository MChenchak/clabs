Напишите программу, запускающую другую программу и отвязывающую ее от управляющего терминала (аналог nohup).
Ввод-вывод запускаемой программы должен быть перенаправлен в /dev/null

### Compilation info:

```
g++ -std=c++17 main.cpp -o run
```

### Usage

```
Syntax: ./run <command>

Examples:
        ./run ls -l
```
